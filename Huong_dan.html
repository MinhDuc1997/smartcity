Hướng dẫn sử dụng api smartcity

API tĩnh (chỉ đọc):

http://techitvn.com/smartcity/api/cauhinh.php

http://techitvn.com/smartcity/api/chedo.php

http://techitvn.com/smartcity/api/congtrinh.php

http://techitvn.com/smartcity/api/congtrinhinfo.php

http://techitvn.com/smartcity/api/denduong.php

http://techitvn.com/smartcity/api/dentag.php

http://techitvn.com/smartcity/api/node.php

http://techitvn.com/smartcity/api/nodeinfo.php

http://techitvn.com/smartcity/api/phanhoi.php

Đăng nhập thông qua http request :

http://techitvn.com/smartcity/api/userlogin.php?name={your_username}&password={your_password}

Ví dụ:

http://techitvn.com/smartcity/api/userlogin.php?name=minhduc&password=01689470862

Nếu đăng nhập thành công json trả về : {“status”:”true”} và ngược lại {“status”:”false”}

API động (Ghi vào csdl)

http://techitvn.com/smartcity/api/action.php?sql={your_query}

Chèn vào một bảng:

Ví dụ:

http://techitvn.com/smartcity/api/action.php?sql=INSERT INTO phanhoi VALUES(‘6′,’duc’,’duc@gmail.com’,’01689470862′,’Ok’,’0′)

nếu INSERT thành công json trả về: {“status”:”true”}, ngược lại nếu dữ lại đã tồn tại hoặc lỗi: {“status”:”false”}

Thêm 1 cột vào bảng:

Ví dụ:

http://techitvn.com/smartcity/api/action.php?sql=ALTER TABLE phanhoi ADD XacNhan int(1)

Cập nhật:

Ví dụ:

http://techitvn.com/smartcity/api/action.php?sql=UPDATE phanhoi SET XacNhan = 1 WHERE XacNhan IS NULL

Xóa:

Ví dụ:

http://techitvn.com/smartcity/api/action.php?sql=ALTER TABLE phanhoi DROP XacNhan
